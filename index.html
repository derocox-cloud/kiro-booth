¿Qué es Kiro?

Kiro es un IDE con inteligencia artificial desarrollado por AWS que va más allá del autocompletado. Kiro te ayuda a pensar, diseñar y construir software de forma más rápida y con mayor confianza.

  • 🤖

    Asistencia con IA

    Chatea con Kiro para explorar ideas, obtener sugerencias de código y resolver dudas sin salir del editor.

  • 📋

    Generación de Specs

    Kiro convierte tus ideas en requisitos, diseño y plan de tareas estructurado — listo para implementar.

  • ✅

    Planificación de Tareas

    Divide proyectos complejos en tareas accionables y deja que Kiro ejecute cada una con contexto completo.

  • ⌨️

    Edición de Código

    Edita, refactoriza y genera código directamente en tus archivos con instrucciones en lenguaje natural.

Cómo descargar Kiro

Sigue estos pasos para instalar Kiro en tu máquina en minutos.

Pasos de instalación

  1. Descarga el instalador para tu sistema operativo desde los enlaces de abajo.
  2. Ejecuta el instalador y sigue las instrucciones en pantalla.
  3. Abre Kiro una vez instalado.
  4. Inicia sesión con tu cuenta de AWS Builder ID (gratuita) o Builder ID existente.
  5. ¡Listo! Abre un proyecto y empieza a trabajar con Kiro.
